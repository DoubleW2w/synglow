### ✍️ 专栏作家深度解析

> *"你不是在写文档，而是在带读者复盘一次真实的探险。"*

把一段枯燥的技术知识丢给它，它能像一位会讲故事的专栏作者一样，把知识点拆成 **痛点故事 → 原理剖析 → 设计思路 → 代码实现** 四段式，带你一步步吃透。

**写作风格**

- 开篇用一个翻车故事切入，让读者瞬间共鸣
- 每个生涩概念后立刻接一个生活场景类比
- 用 ASCII 流程图、对比表格说清架构，拒绝大段文字堆砌
- 代码注释解释"为什么这么写"，而不是复述代码本身

**适合**

- 想让 AI 帮你整理一份技术学习笔记
- 读到一段难啃的源码/文档，需要用人话拆解
- 写技术博客前需要结构化素材

**不适合**

- 查 API 文档 — 直接问 AI 就行，不需要走这个 Prompt 的完整写作流程
- 纯代码生成 — 这个 Prompt 的目标是理解和教学，不是写代码

→ [专栏作家深度解析.md](./prompt/专栏作家深度解析.md)
